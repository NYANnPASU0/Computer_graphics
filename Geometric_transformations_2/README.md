# 2. Геометрические преобразования

Программа демонстрирует афинные преобразования плоских фигур с использованием однородных координат и матричных операций

## 2.1 Преобразования фигуры

Реализованы статистические преобразования геометрической фигуры

![рисунок](images/звезда.png)

Все преобразования реализованы через умножение матрицы 3х3 на вектор однородных координат `[x, y, 1]`

- **Перенос** вдоль оси X и Y
- **Отражение** относительно осей X, Y, и прямой Y = X
- **Масштабирование** (независимо по X и Y)
- **Поворот** относительно центра координат, относительно произвольной точки
- **Восстановление исходного положения фигуры**

## 2.2 Анимация

Реализована простая динамическая анимация, отображающая летящий самолет с вращающимся винтом<br>
Анимация построена на тех матричных преобразованиях, которые были реализованы в пункте 2.1

- Самолет движется слева направо
- При достижении правого края окна возвращается в начальную позицию
- Винт самолета непрерывно вращается за счет последовательных поворотов

**Язык:** Python 3 <br>
**Библиотеки:** Tkinter, ttk, math
