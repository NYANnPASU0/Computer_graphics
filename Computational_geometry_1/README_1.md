# __1. Задачи вычислительной геометрии__

## 1.1 Определение положения двух точек относительно прямой на плоскости

Дана прямая, заданная своими коэффициентами `A`, `B`, `C` (уравнение прямой: `Ax + By + C = 0`), и две точки `M1(x1, y1)`, `M2(x2, y2)`.

Необходимо определить взаимное расположение точек относительно прямой:

- Точки лежат **по одну сторону** от прямой
- Точки лежат **по разные стороны** от прямой
- Одна или обе точки лежат **на прямой**

### Теоретическое обоснование

Для точки с координатами `(x, y)` значение выражения `f(x, y) = Ax + By + C` определяет её положение относительно прямой:

- `f(x, y) > 0` - точка находится с одной стороны прямой
- `f(x, y) < 0` - точка находится с другой стороны прямой
- `f(x, y) = 0` - точка лежит на прямой

**Алгоритм:** <br>
1. Вычисляем `check1 = A*x1 + B*y1 + C` <br>
2. Вычисляем `check2 = A*x2 + B*y2 + C` <br>
3. Сравниваем знаки:

- Если `check1 == 0 или check2 == 0` - точка(и) на прямой
- Если `check1 * check2 > 0` - точки по одну сторону
- Если `check1 * check2 < 0` - точки по разные стороны<br>

## 1.2 Пересечение лучей на плоскости

Точки `A`, `B`, `C`, `D`, лежащие на одной прямой, заданы своими координатами `A(ax, ay)`, `B(bx, by)`, `C(cx, cy)`, `D(dx, dy)`. Определить пересекаются ли лучи `[AB)` и `[CD)`

### Теоретическое обоснование

Луч AB задаётся точкой A и направляющим вектором `v = B - A`

**Скалярное произведение** двух векторов определяется как: <br>
`v × w = |v|*|w|*cos a` <br>
где a - угол между векторами

Знак скалярного произведения показывает ориентацию векторов:

- `v * w > 0` - векторы сонаправлены
- `v * w < 0` - векторы противоположно направлены
- `v * w = 0` - векторы перпендикулярны

Для определения пересечения лучей: <br>
1.Вычисляем скалярное произведение направляющих векторов: `s = v·w` <br>
2.Определяем сонаправленность лучей:

- Если `s > 0` - лучи смотрят в одну сторону - пересекаются
- Если `s < 0` - лучи смотрят в разные стороны - могут пересекаться, если начала лежат между собой

3.Проверяем, лежит ли начало одного луча на продолжении другого через проекции

**Алгоритм:** <br>
1.Вычислить направляющий вектор луча AB: <br>
`vx = bx - ax, vy = by - ay`<br>
2.Вычислить направляющий вектор луча CD: <br>
`wx = dx - cx, wy = dy - cy`<br>
3.Вычислить скалярное произведение: <br>
`sonapr = vx*wx + vy*wy`<br>
4.Вычислить положение точки C относительно луча AB:<br>
`posC = (cx - ax)*vx + (cy - ay)*vy`<br>
5.Вычислить положение точки A относительно луча CD:<br>
`posA = (ax - cx)*wx + (ay - cy)*wy`<br>
6.Если `sonapr > epsilon` - лучи сонаправлены, пересекаются<br>
7.Иначе если `sonapr < -epsilon` - лучи противоположно направлены

- Если `posC >= -epsilon` и `posA >= -epsilon`- лучи пересекаются
- Иначе - лучи не пересекаются<br>

## 1.3 Принадлежность точки углу в пространстве

Точки `A`, `B`, `C`, `D` заданы своими координатами. Выясните, лежит ли точка `D` внутри угла `ABC`? <br>
Имеется ввиду тот из углов `ABC`, который заметается лучом при повороте от `BA` к `BC` против часовой стрелки

### Теоретическое обоснование

**Векторное произведение** двух векторов `A` и `B`:<br>
`A * B = |A|*|B|*sin a·n`<br>
где n — единичный вектор, перпендикулярный плоскости векторов a и b

- A × B перпендикулярен и A, и B
- Если A × B = 0, векторы коллинеарны

**Смешанное произведение** трёх векторов:<br>
`(A × B)×C`

- Если смешанное произведение равно нулю, векторы компланарны (лежат в одной плоскости)

1.Проверяем, лежат ли все четыре точки в одной плоскости (смешанное произведение = 0).Если нет - точка D заведомо не может лежать внутри угла <br>
2.Если точки компланарны, проверяем, находится ли точка D между лучами BA и BC <br>

- Точка D лежит внутри угла, если при обходе B - A - D и B - D - C повороты имеют одинаковое направление (против часовой стрелки)

`sign1 = (BA × BD)·n` — определяет положение D относительно BA <br>
`sign2 = (BC × BD)·n` — определяет положение D относительно BC <br>

**Алгоритм:** <br>
1.Вычислить векторы:<br>
`BA = A - B`<br>
`BC = C - B`<br>
`BD = D - B`<br>
2.Вычислить нормаль к плоскости угла:<br>
`n = BA × BC`<br>
3.Вычислить смешанное произведение для проверки компланарности:<br>
`compl = n · BD`<br>
4.Если `|compl| > epsilon` - точки не компланарны, D вне угла<br>
5.Вычислить первый знак:<br>
`sign1 = (BA × BD) · n`<br>
6.Если `sign1 <= -epsilon` - точка вне угла<br>
7.Вычислить второй знак:<br>
`sign2 = (BC × BD) · n`<br>
8.Если `sign2 >= epsilon` - точка вне угла"<br>
9.Иначе - точка внутри угла<br>

## 1.4 Положение двух точек относительно плоскости в пространстве

Дана прямая, заданная своими коэффициентами `A`, `B`, `C`, `D` (уравнение прямой: `Ax + By + Cz + D = 0`), и две точки `M1(x1, y1, z1)`, `M2(x2, y2, z2)`.

Необходимо определить взаимное расположение точек относительно плоскости:

- Точки лежат **по одну сторону** от плоскости
- Точки лежат **по разные стороны** от плоскости
- Одна или обе точки лежат **на плоскости**

### Теоретическое обоснование

Для произвольной точки `M(x, y, z)` значение функции `F(x, y, z) = Ax + By + Cz + D` определяет положение точки относительно плоскости:

- `F(x, y, z) = 0` - точка лежит на плоскости
- `F(x, y, z) > 0` - точка находится с одной стороны плоскости
- `F(x, y, z) < 0` - точка находится с другой стороны плоскости

**Алгоритм:** <br>
1.Вычислить `check_1 = A*x1 + B*y1 + C*z1 + D` <br>
2.Вычислить `check_2 = A*x2 + B*y2 + C*z2 + D` <br>
3.Если `|check_1| < epsilon` и `|check_2| < epsilon` - обе точки лежат на плоскости <br>
4.Иначе если `|check_1| < epsilon`- первая точка лежит на плоскости <br>
5.Иначе если `|check_2| < epsilon`- вторая точка лежит на плоскости <br>
6.Иначе если `check_1 * check_2` > 0 - точки лежат по одну сторону от плоскости <br>
7.Иначе - точки лежат по разные стороны от плоскости <br>
