# __1. Задачи вычислительной геометрии__

## 1.1 Определение положения двух точек относительно прямой на плоскости

Дана прямая, заданная своими коэффициентами `A`, `B`, `C` (уравнение прямой: `Ax + By + C = 0`), и две точки `M1(x1, y1)`, `M2(x2, y2)`.

Необходимо определить взаимное расположение точек относительно прямой:

- Точки лежат **по одну сторону** от прямой
- Точки лежат **по разные стороны** от прямой
- Одна или обе точки лежат **на прямой**

### Теоретическое обоснование

Для точки с координатами `(x, y)` значение выражения `f(x, y) = Ax + By + C` определяет её положение относительно прямой:

- `f(x, y) > 0` - точка находится с одной стороны прямой
- `f(x, y) < 0` - точка находится с другой стороны прямой
- `f(x, y) = 0` - точка лежит на прямой

**Алгоритм:** <br>
1. Вычисляем `check1 = A*x1 + B*y1 + C` <br>
2. Вычисляем `check2 = A*x2 + B*y2 + C` <br>
3. Сравниваем знаки:

- Если `check1 == 0 или check2 == 0` - точка(и) на прямой
- Если `check1 * check2 > 0` - точки по одну сторону
- Если `check1 * check2 < 0` - точки по разные стороны

## 1.2 Пересечение лучей на плоскости

Точки `A`, `B`, `C`, `D`, лежащие на одной прямой, заданы своими координатами `A(ax, ay)`, `B(bx, by)`, `C(cx, cy)`, `D(dx, dy)`. Определить пересекаются ли лучи `[AB)` и `[CD)`

### Теоретическое обоснование

Луч AB задаётся точкой A и направляющим вектором `v = B - A`.

Скалярное произведение двух векторов определяется как: <br>
`v · w = |v|·|w|·cos a` <br>
где a - угол между векторами.

Знак скалярного произведения показывает ориентацию векторов:

- `v · w > 0` - векторы сонаправлены
- `v · w < 0` - векторы противоположно направлены
- `v · w = 0` - векторы перпендикулярны

Для определения пересечения лучей: <br>
1.Вычисляем скалярное произведение направляющих векторов: `s = v·w` <br>
2.Определяем сонаправленность лучей:

- Если `s > 0` - лучи смотрят в одну сторону - пересекаются
- Если `s < 0` - лучи смотрят в разные стороны - могут пересекаться, если начала лежат между собой

3.Проверяем, лежит ли начало одного луча на продолжении другого через проекции.

**Алгоритм:** <br>
1.Вычислить направляющий вектор луча AB: <br>
`vx = bx - ax, vy = by - ay`<br>
2.Вычислить направляющий вектор луча CD: <br>
`wx = dx - cx, wy = dy - cy`<br>
3.Вычислить скалярное произведение: <br>
`sonapr = vx*wx + vy*wy`<br>
4.Вычислить положение точки C относительно луча AB:<br>
`posC = (cx - ax)*vx + (cy - ay)*vy`<br>
5.Вычислить положение точки A относительно луча CD:<br>
`posA = (ax - cx)*wx + (ay - cy)*wy`<br>
6.Если `sonapr > epsilon` - лучи сонаправлены, пересекаются<br>
7.Иначе если `sonapr < -epsilon` - лучи противоположно направлены

- Если `posC >= -epsilon` и `posA >= -epsilon`- лучи пересекаются
- Иначе - лучи не пересекаются
